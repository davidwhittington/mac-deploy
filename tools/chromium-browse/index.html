Traffic Generation

Browse like a human.
Through any Chromium browser.

chromium-browse is a headless traffic generator that drives any Chromium-based browser via its native DevTools Protocol. Load a URL list, randomize visits, follow links, simulate scroll and dwell behavior. No Puppeteer, no Playwright, no third-party engine.

🌐 Uses the real browser. Traffic comes from the actual browser binary you select. Enterprise browsers like Island, Comet, and Atlas send their real fingerprint and policy headers.
Install via Homebrew
brew tap davidwhittington/mac-security
brew install chromium-browse
Requires macOS with Python 3.12. Dependencies (websockets, aiohttp) install automatically.

Supported browsers

Any Chromium-based browser that supports the --remote-debugging-port flag. Auto-detects install paths on macOS, Linux, and Windows.

How it works

No browser automation framework sits between you and the browser. chromium-browse talks directly to the browser's built-in Chrome DevTools Protocol over a local WebSocket.

📄

URL list input

Reads from a plain text file. One URL per line. Comments and blanks ignored.

🔀

Randomized order

Shuffles the URL list each round. No predictable access pattern.

🔗

Link crawling

Follows same-domain links up to a configurable depth. Skips assets and off-site URLs.

📜

Scroll simulation

Random scroll positions on each page to trigger lazy-loaded content and analytics.

⏱️

Dwell time

Randomized pause on each page between configurable min/max bounds.

🔄

Multi-round

Run multiple passes through the list. Each round re-shuffles independently.

Options

--browser, -b Browser name (island, chrome, brave, etc.) or absolute path to any Chromium binary
--urls Path to URL list file (default: urls.txt)
--max-depth How many levels deep to follow links per site (default: 2)
--max-links Max links to follow per page (default: 5)
--min-delay Minimum dwell time in seconds (default: 2)
--max-delay Maximum dwell time in seconds (default: 8)
--rounds Number of full passes through the URL list (default: 1)
--list-browsers Show all known browsers and whether they're installed, then exit
--cdp-port Chrome DevTools Protocol port (default: 9222)

Quick start

# 1. Install
brew tap davidwhittington/mac-security
brew install chromium-browse

# 2. Create a URL list
cp "$(brew --prefix chromium-browse)/share/chromium-browse/urls.txt.example" ~/urls.txt
# Edit ~/urls.txt with your target URLs

# 3. Run with Island browser
chromium-browse --browser island --urls ~/urls.txt

# 4. Or use Chrome with more rounds and deeper crawling
chromium-browse --browser chrome --rounds 5 --max-depth 3

# 5. Check what browsers are available
chromium-browse --list-browsers
