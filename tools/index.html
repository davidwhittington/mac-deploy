Security hardening and auditing tools ship through the mac-security tap. Privacy and utility tools have moved to mac-tools.

Add the tap
brew tap davidwhittington/mac-security
macOS Ventura / Sonoma / Sequoia / Tahoe · Apple Silicon and Intel

Security tools

Security hardening and auditing for macOS workstations.

mac-securityv0.10.0
Full security posture audit and hardening toolkit for macOS workstations. FileVault, SIP, Gatekeeper, firewall, SSH config, open ports, sharing services, user accounts, and update policy. Outputs structured Markdown reports.
security audit hardening
brew install mac-security

Also see: mac-tools

Privacy, maintenance, and development utilities have moved to a dedicated repo and tap.

tor-proxy
Route macOS system traffic through the Tor network via SOCKS proxy.
privacy
brew install davidwhittington/mac-tools/tor-proxy
chromium-browse
Headless browsing traffic generator for any Chromium-based browser via CDP.
network automation
brew install davidwhittington/mac-tools/chromium-browse
brew-upgrade
Automated Homebrew package updater with change logging.
maintenance
brew install davidwhittington/mac-tools/brew-upgrade
setup-claude
Claude Code installation and configuration for new Macs.
dev environment
brew install davidwhittington/mac-tools/setup-claude

Installing

Security tools and utility tools use separate taps.

Examples
# Security hardening and auditing
brew tap davidwhittington/mac-security
brew install davidwhittington/mac-security/mac-security

# Utility and privacy tools
brew tap davidwhittington/mac-tools
brew install davidwhittington/mac-tools/tor-proxy
brew install davidwhittington/mac-tools/chromium-browse
